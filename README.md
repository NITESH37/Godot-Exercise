# Godot-Exercise

## Important Steps to Follow 

### Step-1 : Check Status  
 `git status`
 
### Step-2 :  Stage Files
  `git add .`
  
### Step-3 : Commit
   `git commit -m "your commit message"`
   
### step-4 : pull from remote repo
	`git pull origin <BRANCH NAME>`
	
### Step-5 : Push to remote repo
    `git push origin <BRANCH NAME>`
